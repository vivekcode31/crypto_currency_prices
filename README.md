# Cryptocurrency Prices Application

## Overview
This application is designed to display live cryptocurrency prices using JavaScript, HTML, and CSS. It leverages the CoinGecko API (or similar cryptocurrency API) to provide real-time data for users.

## Features
- Live price updates for Bitcoin and other major cryptocurrencies.
- User-friendly interface with HTML and CSS for a seamless experience.
- Responsive design to ensure usability on various devices.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **API:** CoinGecko API (or alternative)

## Installation
To run this application locally, follow these steps:
1. Clone the repository:
   ```bash
git clone https://github.com/vivekcode31/crypto_currency_prices.git
   ```
2. Navigate into the project directory:
   ```bash
cd crypto_currency_prices
   ```
3. Open `index.html` in your web browser to view the application.

## Usage
Simply open the application in a web browser to see the live prices of various cryptocurrencies. The application fetches data from the CoinGecko API and displays it in a user-friendly format.
